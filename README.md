# Joyful Folio Forge

A personal portfolio built with Next.js (App Router), React, TypeScript, Tailwind CSS, and shadcn/ui.

## Tech Stack

- Next.js
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`.

## SEO Setup

- Metadata is defined in `app/layout.tsx`.
- `robots.txt` is generated by `app/robots.ts`.
- `sitemap.xml` is generated by `app/sitemap.ts`.
- Set `NEXT_PUBLIC_SITE_URL` in your environment for correct canonical/sitemap host.

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - lint project
